PHP и JavaScript

Языки программирования PHP и JavaScript позволяет сделать сайт динамичным, то есть реагирующим на действия пользователя. Например, можно сделать красивую выпадающую менюшку или слайдер

Виды скриптов

Для этого пишутся скрипты (англ. script - "сценарий") - программы, позволяющие реагировать на действие пользователя. Скрипты бывают двух видов:

  • те, которые выполняються на сервере, а результат их выполнения проходит в браузер к пользователю уже в готовом виде. Это скрипты, написаные на языке PHP. На нем пишутся CMS-ки - системы управления контентом.

  • те, которые выполняються прямо в браузере пользователя. Это скрипты, написаные на языке JavaScript. Они чаще всего используються для, того чтобы сделать страницу более удобной и красивой.
